# Práctica universitaria de C# (2022): Mecánicas de plataformas (Unity).
01/2022

## Qué hace cada script
- CheckpointSystem.cs: guarda la posición del jugador al pulsar la barra espaciadora y lo teletransporta de vuelta a ese punto si toca un elemento concreto (el agua).
- QuestTrigger.cs: es como funciona la lógica de los interruptores: activa un objeto, desactiva otro y reproduce un sonido cuando el jugador pisa un trigger.
- TimeTrialManager.cs: controla el cronómetro del minijuego, gestiona la música de fondo, el sonido del reloj y permite aumentar el tiempo límite si el jugador recoge el objeto de tiempo extra.
